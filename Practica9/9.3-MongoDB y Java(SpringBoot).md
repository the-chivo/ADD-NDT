# Tema9.3 MongoDB Atlas y Java (SpringBoot)

## 1. Creacion de un proyecto SpringBoot + MongoDB
* Ir a https://start.spring.io/
* Añadir la dependencia de Spring Data MongoDB

## 2. Creación de usuario en MongoDB atlas

* Registrarse en https://account.mongodb.com/account/login
* En el proceso de creación de un uruasrop se asignara un cluster para la DB
* Ademas se nos otorgara un string de conexion para conectar la base de datos con nuestra aplicacion.

## 3. Conexion con MongoDB Atlas

#### El string de conexion tiene la siguiente estructura:
mongodb+srv://<usuario>:<contraseña>@<cluster>.mongodb.net/<nombreBaseDatos>?retryWrites=true&w=majority
#### Debemos añadir esto a la propiedad "spring.data.mongodb.uri" del archivo Application.properties de SpringBoot
#### Al lanzar la aplicacion ya estaremos concetados a nuestra base de datos en la nube MongoDB Atlas
